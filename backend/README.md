# RezerveMe Backend

Spring Boot backend application for RezerveMe.

## Prerequisites

- Java 17 or higher
- Maven 3.6+
- PostgreSQL database

## Setup

1. Create a PostgreSQL database named `rezerveme`
2. Update database credentials in `src/main/resources/application.properties` or `application.yml`
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```

## API

The API will be available at `http://localhost:8080`
